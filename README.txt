もともとVC++2008で作られていたプロジェクトをVC++2010でコンパイルしなおしています。

外部ライブラリとして、DXライブラリ（http://homepage2.nifty.com/natupaji/DxLib/）を使っており、
このプロジェクトをコンパイルするには、DXライブラリをダウンロードした後、

	「構成プロパティ」->「Ｃ／Ｃ＋＋」->「全般」->「追加のインクルードディレクトリ」および
	「構成プロパティ」->「リンカー」->「全般」->「追加のライブラリディレクトリ」

に、「(DXライブラリのダウンロード先)\DxLib_VC\プロジェクトに追加すべきファイル_VC用」を追加する必要があります
